Versions
========

Simpletask Revived currently keeps multiple source flavors, but the primary revived release path is the cloudless build.

Simpletask Revived Cloudless
----------------------------

-   Stores tasks in a local `todo.txt` file on the device.
-   Works well with external sync tools such as Syncthing.
-   Is the primary flavor targeted for the first revived F-Droid submission.

Remote-sync flavors
-------------------

-   Nextcloud and WebDAV remain in the source tree.
-   They are being kept coherent, but they are not the primary release path for the first revived fork release.

Dropbox
-------

-   Dropbox support remains in the repository.
-   It is not part of the initial F-Droid submission path for Simpletask Revived.
